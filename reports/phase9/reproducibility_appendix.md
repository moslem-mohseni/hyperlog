## 9. Table 5 — Eight-axis ablation (§5.4, `tab:ablation`)

Source: `reports/phase6/runs/<run>/ablation_matrix.csv`. One row per
(axis, variant) pair against the axis baseline:

| CSV column | Paper cell |
|---|---|
| `axis` | Axis name (A1–A8) |
| `variant` | Variant column |
| `delta_mean` | $\Delta$F1 (vs. baseline) |
| `cliffs_delta` | Cliff's $\delta$ |
| `wilcoxon_p_holm` | $p$ (Holm) |
| `significant_under_holm` | sig. checkmark |

The joint significance criterion (`significant_under_holm`) is
`wilcoxon_p_holm < 0.05 AND |cliffs_delta| > 0.33`.

Per-axis Markdown summaries are in
`reports/phase6/runs/<run>/<axis>/<axis>.md`.

## 11. Reviewer command sheet

```powershell
# 1. Clone and install.
git clone https://github.com/moslem-mohseni/hyperlog.git
cd hyperlog
pip install -r requirements-lock.txt
pip install -e .

# 2. CPU regression — every claim that does NOT need a GPU.
.\scripts\reproduce_all.ps1
#    -> reports/phase7/reproduction.json
#    -> 420 unit + integration tests pass

# 3. Build the paper.
.\scripts\build_paper.ps1
#    -> paper/figures/*.pdf
#    -> paper/main.pdf

# 4. (Optional) GPU run on a 24 GB GPU — fills the TBD cells in
#    paper/main.tex with real numbers.
hylog-train --config configs/experiments/hylog_hdfs.yaml
hylog-loso --config configs/experiments/loso_hdfs_held.yaml
hylog-calibrate --predictions reports/phase4/runs/<run>/hdfs/predictions.jsonl --out-dir reports/phase5/runs/hdfs
hylog-ablation --all-axes configs/ablation --out-dir reports/phase6/runs
```

Every TBD in the paper corresponds to exactly one JSON file in this
appendix; once the GPU artefact exists, `scripts/build_figures.py` and
the LaTeX source pick it up without manual editing.
